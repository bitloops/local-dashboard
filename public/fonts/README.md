# Self-hosted fonts

Fonts are loaded via `@font-face` in `src/styles/index.css` from this directory. Add the following `.woff2` files:

**Montserrat** (default UI font)
- Montserrat-Regular.woff2 (400)
- Montserrat-Medium.woff2 (500)
- Montserrat-SemiBold.woff2 (600)
- Montserrat-Bold.woff2 (700)

**Inter**
- Inter-Regular.woff2 (400)
- Inter-Medium.woff2 (500)
- Inter-SemiBold.woff2 (600)
- Inter-Bold.woff2 (700)

**Manrope**
- Manrope-Regular.woff2 (400)
- Manrope-Medium.woff2 (500)
- Manrope-SemiBold.woff2 (600)
- Manrope-Bold.woff2 (700)

You can download these from the [fontsource](https://fontsource.org) packages and copy the woff2 files into this folder with the names above.
